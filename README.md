# realtime-blockchain-payroll-system-using-algorand
Real-time programmable payroll system using Algorand blockchain for continuous salary streaming and automated payment execution.

-> Real-Time Blockchain Payroll System using Algorand

-> Project Overview

This project implements a decentralized real-time payroll infrastructure using Algorand blockchain. It enables organizations to stream salaries continuously, automate milestone payments, and ensure transparent financial transactions.

-> Features

* Real-time salary streaming
* Tokenized payroll system
* Milestone-based payments
* Wallet integration
* Transaction history tracking
* Delegated access control
* Blockchain payment using Algorand
* Dashboard UI for employee payroll

-> Tech Stack

Frontend:

* HTML
* Tailwind CSS
* JavaScript

Backend:

* Python
* Flask
* SQLite Database

Blockchain:

* Algorand SDK

-> Setup Instructions

1. Clone repository

2. Install dependencies
   pip install -r requirements.txt

3. Run database setup
   python database_setup.py

4. Start backend server
   python app.py

5. Open frontend/index.html in browser
